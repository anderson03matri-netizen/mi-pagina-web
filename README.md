# Proyecto de Presentación Personal en HTML5

### Propósito
El objetivo de este proyecto es demostrar el dominio de la estructura semántica de HTML5, creando una página de presentación personal accesible y bien organizada.

### Aplicación de Semántica y Accesibilidad
1. **Semántica:** Se utilizaron etiquetas como `<header>`, `<main>`, `<section>` y `<footer>` para definir áreas claras de contenido, facilitando el SEO y la lectura por parte de navegadores.
2. **Accesibilidad:** 
   - Se incluyeron atributos `alt` en imágenes.
   - Se garantizó que los formularios sean legibles para lectores de pantalla usando `<label>`.
   - Se añadió navegación mediante enlaces internos para facilitar el uso con teclado.
3. **Comentarios:** El código fuente contiene notas explicando la función de cada bloque principal.
